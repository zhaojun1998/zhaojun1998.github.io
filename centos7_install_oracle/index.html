你可以直接复制以下内容内容进行覆盖, 不过以下参数中有几个需要自己配置下:

  • ORACLE_HOSTNAME: 应该是你的主机名称, 通过命令 hostname 获取.

  • oracle.install.db.config.starterdb.globalDBName: 指定 GlobalName

  • oracle.install.db.config.starterdb.SID: 指定 SID

  • oracle.install.db.config.starterdb.characterSet: 字符集, 通常中文选择 ZHS16GBK 简体中文库. 根据自己的需要来配置.

  • oracle.install.db.config.starterdb.password.ALL: 所有自带用户的密码

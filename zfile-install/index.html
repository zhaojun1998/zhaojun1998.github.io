安装依赖

域名访问

域名访问可直接将域名 A 记录解析到服务器 IP, 即可通过 域名:端口 访问.

如需直接使用域名, 不带端口, 可将程序设置为 80 端口, 或者使用 nginx 或 caddy 等工具进行反向代理.

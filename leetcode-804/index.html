LeetCode 804 Unique Morse Code Words

发表于 2018-11-19 更新于 2018-11-28 分类于 算法

题意

给予一个编码表，26 个字母分别对应一个编码，给定一组单词，获取单词的所有字母组合后的编码中不重复的数量。

解法

首先为每个单词的每个字符进行转码, 将转码后的数据放到 Set 集合中, 最后返回 Set 的长度。

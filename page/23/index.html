合并排序数组 Ⅱ

发表于 2017-07-29 更新于 2018-07-06 分类于 算法

题意

合并两个排序的整数数组 A 和 B 变成一个新的数组。

注意事项：你可以假设A具有足够的空间（A数组的大小大于或等于 m + n）去添加B中的元素。

ps：m 表示 A 数组的有效元素个数，n 代表 B 数组的有效元素个数。

样例

给出 A = [1, 2, 3, empty, empty], B = [4, 5]

合并之后 A 将变成 [1,2,3,4,5]

移动零

发表于 2017-08-08 更新于 2018-07-06 分类于 算法

题意

给一个数组 nums 写一个函数将 0 移动到数组的最后面，非零元素保持原数组的顺序

注意事项：

  1. 必须在原数组上操作
  2. 最小化操作数

样例

给出 nums = [0, 1, 0, 3, 12], 调用函数之后, nums = [1, 3, 12, 0, 0].

思路

设两个指针，先都指向数组头元素，一个依次向后找不为零的数，然后与另一个指向的数进行交换，直到找到最后一个元素为止。
